⚠ Peraturan Kode Darurat:
Kode berlaku hanya hari ini dan hanya sekali pakai. Guru tetap wajib berada dalam area GPS sekolah. Berikan kode secara langsung atau via pesan pribadi — jangan posting ke grup umum.
🔧 Perlu konfigurasi Firebase Security Rules
Buka Firebase Console → Realtime Database → Rules, lalu tambahkan rule berikut di dalam "rules":

"kode_darurat": { "$tgl": { ".read": "auth != null && root.child('users').child(auth.uid).child('role').val() === 'admin'", "$kode": { ".write": "auth != null" } } }
